## Contributing
### Code formatting
```sh
npm run format:fix
```

### Adding dependency
Adding dependency to the whole project (web + mobile)
```sh
npm i <package-name>
```

Adding dependency to a specific workspace
```sh
npm i <package-name> --workspace <workspace-name>
```

Adding dev dependency
```sh
npm i -D <package-name>
```
